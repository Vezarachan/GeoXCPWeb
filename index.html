SHAP Predictor MLP Regression
A multi-layer perceptron (MLP) is trained to predict SHAP values from house features and model output. Because SHAP values have no observable ground truth—they describe model behavior, not physical reality—the MLP output Ê[SHAP] serves as the reference (expected) SHAP value.

What is Spatial Explanation Uncertainty?
The reliability of a SHAP value at any location is jointly affected by three sources:
  • Data — spatial heterogeneity and sampling gaps in training data
  • Prediction model — XGBoost sensitivity to local feature patterns
  • Explainability model — MLP regression error when estimating SHAP
GeoXCP wraps geographically-weighted conformal prediction around the SHAP Predictor to produce location-aware intervals that absorb all three sources, with a formal 90% coverage guarantee.
